# Tooling and Resources
Here are some of the main developer tools to get you started developing on Sei, separated into Cosmos and EVM components

## Cosmos
- **[Flipside](https://flipsidecrypto.xyz/):** Data analytics platform for Sei
- **TODO: RPC**
- **[Sei-js](https://www.npmjs.com/package/@sei-js/core):** Sei specific javascript library
- **[Cosmwasm](https://docs.cosmwasm.com/docs/):** Primary smart contract platform for cosmos
- **[Cosmovisor](https://docs.cosmos.network/main/build/tooling/cosmovisor):** process manager for Cosmos SDK
applications to allow for automated chain upgrades
- **[Protobuf](https://protobuf.dev/):** data serialization/deserialization data format used extensively in Cosmos
- **[Awesome Cosmos](https://github.com/cosmos/awesome-cosmos):** Collection of other popular resources in Cosmos
