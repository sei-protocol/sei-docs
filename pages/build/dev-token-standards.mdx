## Fungible Tokens

**ERC20 Standard**  
Defines interchangeable tokens for EVM-based dApps.

**CW20 Standard**  
Cosmos equivalent of ERC20 for compatibility with Cosmos dApps.

**Cross-Chain Compatibility**  
Enables seamless cross-chain interaction through pointer contracts.

## NFTs

**ERC721 Standard**  
Defines unique tokens for EVM-based chains.

**CW721 Standard**  
Cosmos equivalent of ERC721 for Cosmos-based dApps.

**Royalty Standards**  
Supports ERC2981 and CW2981 for creator royalties.

**Cross-Chain Compatibility**  
Pointer contracts enable cross-standard NFT interactions.
