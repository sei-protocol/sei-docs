# APIs
Once you have access to an RPC endpoint, either your own node or one from our
[registry](https://github.com/sei-protocol/chain-registry/blob/main/chains.json), you are ready to interact with the
chain.

## Sei Endpoints
All Cosmos queries and transactions can be made via gRPC (default port: 9090), REST (default port: 1317) or RPC
(default port: 26657). Below are queries for an account's bank balances in each of the 3 ways. Please consult the
official [Cosmos SDK docs](https://docs.cosmos.network/) for more APIs.

### Query balances gRPC
```
// AllBalances queries the balance of all coins for a single account.
//
// When called from another module, this query might consume a high amount of
// gas if the pagination field is incorrectly set.
rpc AllBalances(QueryAllBalancesRequest) returns (QueryAllBalancesResponse) {
  option (cosmos.query.v1.module_query_safe) = true;
  option (google.api.http).get               = "/cosmos/bank/v1beta1/balances/{address}";
}
```

### Query balances REST
```
GET /cosmos/bank/v1beta1/balances/{address}
```

### Query balances RPC
```
seid query bank balances {address}
```
