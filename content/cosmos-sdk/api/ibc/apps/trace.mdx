# denom_hashes/(trace)

## Endpoint
```http
get /ibc/apps/transfer/v1/denom_hashes/{trace}
```

## Parameters
> **Request Parameters**

- **trace** (path): The denomination trace ([port_id]/[channel_id])+/[denom]

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
