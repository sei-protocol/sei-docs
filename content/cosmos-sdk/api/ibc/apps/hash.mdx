# denom_traces/(hash)

## Endpoint
```http
get /ibc/apps/transfer/v1/denom_traces/{hash}
```

## Parameters
> **Request Parameters**

- **hash** (path): hash (in hex format) or denom (full denom with ibc prefix) of the denomination trace information.

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
