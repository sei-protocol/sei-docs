# (packet_commitment_sequences)/unreceived_packets

## Endpoint
```http
get /ibc/core/channel/v1/channels/{channel_id}/ports/{port_id}/packet_commitments/{packet_commitment_sequences}/unreceived_packets
```

## Parameters
> **Request Parameters**

- **channel_id** (path): channel unique identifier
- **port_id** (path): port unique identifier
- **packet_commitment_sequences** (path): list of packet sequences

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
