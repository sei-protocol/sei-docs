# params/(params_type)

## Endpoint
```http
get /cosmos/gov/v1beta1/params/{params_type}
```

## Parameters
> **Request Parameters**

- **params_type** (path): params_type defines which parameters to query for, can be one of "voting",
"tallying" or "deposit".

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
