# (delegator_address)/rewards

## Endpoint
```http
get /cosmos/distribution/v1beta1/delegators/{delegator_address}/rewards
```

## Parameters
> **Request Parameters**

- **delegator_address** (path): delegator_address defines the delegator address to query for.

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
