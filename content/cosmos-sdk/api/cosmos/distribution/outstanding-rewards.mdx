# (validator_address)/outstanding_rewards

## Endpoint
```http
get /cosmos/distribution/v1beta1/validators/{validator_address}/outstanding_rewards
```

## Parameters
> **Request Parameters**

- **validator_address** (path): validator_address defines the validator address to query for.

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
