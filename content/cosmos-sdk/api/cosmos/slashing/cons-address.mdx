# signing_infos/(cons_address)

## Endpoint
```http
get /cosmos/slashing/v1beta1/signing_infos/{cons_address}
```

## Parameters
> **Request Parameters**

- **cons_address** (path): cons_address is the address to query signing info of

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
