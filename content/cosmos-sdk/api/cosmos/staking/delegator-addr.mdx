# delegations/(delegator_addr)

## Endpoint
```http
get /cosmos/staking/v1beta1/validators/{validator_addr}/delegations/{delegator_addr}
```

## Parameters
> **Request Parameters**

- **validator_addr** (path): validator_addr defines the validator address to query for.
- **delegator_addr** (path): delegator_addr defines the delegator address to query for.

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
