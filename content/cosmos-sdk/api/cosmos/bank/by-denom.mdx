# (address)/by_denom

## Endpoint
```http
get /cosmos/bank/v1beta1/balances/{address}/by_denom
```

## Parameters
> **Request Parameters**

- **address** (path): address is the address to query balances for.
- **denom** (query): denom is the coin denom to query balances for.

## Responses
> **Possible Responses**

- **200**: A successful response.
- **default**: An unexpected error response.
